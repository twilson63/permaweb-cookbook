# React 入门套件

React 是一个用于构建用户界面的流行库。除了像 create-react-app 等其他流行工具之外，React 项目可以编译成一个 bundle（打包文件）。该 bundle 可以作为交易上传到 Permaweb，在那里它会作为单页应用程序 (SPA) 提供。

React 入门套件指南：

- [Vite](./turbo.md) - React + Vite，使用 `permaweb-deploy` 发布

::: info Permaweb 应用限制

- 100% 前端应用程序（不包含服务器端后端）
- 应用程序会从子路径提供（https://[gateway]/[TX_ID]）
  :::
