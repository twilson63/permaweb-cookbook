# 去中心化计算

协议本身并未直接内建任何执行计算的机制——然而，这并不表示开发者无法在 Arweave 上创建并存储智能合约。

[AO](https://ao.arweave.net) 是一个建立在 Arweave 之上的去中心化运算网络。通过使用 Arweave 作为存储智能合约状态与用户交互的状态层，可以构建复杂的去中心化应用程序。

说明 AO 的运行原理与如何使用它来开发应用程序并非本手册的范围，但若想进一步学习，仍有大量资源可用于入门。
