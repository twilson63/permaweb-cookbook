# Permaweb로 요리하기

Permaweb 요리책은 Arweave 위에 애플리케이션을 구축하기 위한 필수 개념과 참고 자료를 제공하는 개발자 리소스입니다. 각 개념과 참고 자료는 Permaweb 개발 생태계의 특정 측면에 초점을 맞추고 추가 세부사항과 사용 예제를 제공합니다.

특히 스마트 계약을 생성하는 데 관심이 있는 분들은 AO 위에 구축된 탈중앙화 컴퓨팅 플랫폼을 다루는 [AO 요리책](https://cookbook_ao.arweave.net)을 확인해 보시기 바랍니다.

## 개발자

영원히 블록체인에 새겨지는 과거와 무한한 가능성이 열려 있는 미래가 공존하는 Arweave 개발 커뮤니티에 오신 것을 환영합니다. 함께 탈중앙화 웹을 구축해 봅시다!

[자세히 읽기](welcome.md)

## 기여하기

요리책은 새로운 Permaweb 개발자가 기여하기 쉽도록 설계되어 있습니다. 어떤 것을 하는 방법을 모른다 하더라도, 요리책에 기여하는 것은 배우기에 좋은 방법입니다!

모든 오픈 이슈는 [여기](https://github.com/twilson63/permaweb-cookbook/issues)에서 확인하세요. 기여 가이드는 [여기](https://github.com/twilson63/permaweb-cookbook/blob/main/CONTRIBUTING.md)에 있습니다. 요리책에 개념, 가이드 또는 참고 자료가 누락되어 있다면 이슈를 추가해 주세요.

[자세히 읽기](contributing.md)

## 요리책 읽는 법

Permaweb 요리책은 서로 다른 목표를 가진 여러 섹션으로 나뉘어 있습니다.

| 섹션                      | 설명                                                                   |
| ------------------------- | ---------------------------------------------------------------------- |
| 핵심 개념 (Core Concepts) | 개발에 알아두면 좋은 Permaweb의 구성 요소                              |
| 가이드 (Guides)           | 개발을 위한 다양한 도구에 대한 간단한 가이드                           |
| 참조 (References)         | 자주 필요로 하는 코드 스니펫에 대한 참조                               |
| 스타터 킷 (Starter Kits)  | Permaweb에서 빠르게 개발을 시작할 수 있는 프론트엔드 프레임워크 스타터 |
