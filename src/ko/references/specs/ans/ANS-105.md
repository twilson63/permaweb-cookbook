# [ANS-105: 라이선스 태그](https://github.com/ArweaveTeam/arweave-standards/blob/master/ans/ANS-105.md)

저자: Sam Williams <sam@arweave.org>, Abhav Kedia <abhav@arweave.org>

## 요약

이 표준은 퍼마웹(permaweb)에 게시된 임의의 트랜잭션에 대해 해당 데이터가 게시된 라이선스를 태그로 명시하는 메커니즘을 정의합니다. 이 표준은 퍼마웹 생태계 전반에 걸친 널리 채택을 장려하기 위해 단순성과 사용 편의성에 중점을 둡니다.

## 동기

Arweave 생태계의 목표는 모든 가치 있는 지식과 역사의 영구적이고 집합적인 공유 자원을 구성하여 언제든지 모든 사람이 접근할 수 있도록 만드는 것입니다. 원래의 웹과 마찬가지로 이 개방형 생태계는 데이터의 광범위한 재사용과 복제를 장려합니다. 전통적인 웹에서는 이로 인해 라이선스 데이터가 부재한 채 널리 공유되는 정보의 방대한 말뭉치가 형성되었습니다.

Arweave 트랜잭션과 데이터 항목이 제공하는 태깅 시스템을 간단히 활용함으로써 우리는 이러한 상황의 반복을 피하고, 대신 모든 데이터 조각의 라이선스를 전달하는 것이 일반적인 관행인 웹을 구축할 수 있습니다. Arweave 태그가 관련 데이터와 분리할 수 없이 결합되어 있다는 특성 때문에, 준수하는 모든 트랜잭션의 라이선스 데이터는 _원자적으로_ 사용처에 전달됩니다 — 데이터로의 링크를 전달하면서 동시에 그 라이선스 정보를 전달하지 않는 것은 불가능합니다. Arweave의 태깅 시스템 특성상, 이는 사용자 경험(데이터가 브라우저에서 렌더링되는 경우 등)에 필수적이거나 눈에 띄는 영향을 주지 않고 달성됩니다.

## 명세

라이선스 태깅 형식은 Arweave 네트워크의 모든 트랜잭션에 추가되는 단일 태그와, 라이선스 유형을 정의하기 위한 두 가지 추가 트랜잭션 형식으로 구성됩니다.

### 트랜잭션 데이터에 라이선스 할당

주어진 트랜잭션 형식 하에 트랜잭션의 데이터를 게시하려면 다음 태그들을 추가하십시오:

| 태그 이름 | _선택 사항?_ | 태그 값                                                             |
| --------- | ------------ | ------------------------------------------------------------------- |
| License   | False        | TXID of `License Definition`                                        |
| Title     | True         | The title of the work                                               |
| Creator   | True         | The name/identifier of the creator(s) of the work                   |
| Source    | True         | A link to the source where the material was obtained, if applicable |

### 라이선스 어설션

이전에 업로드된 트랜잭션이 특정 라이선스를 갖는다고 주장하려면 다음 태그들을 제공하십시오:
| 태그 이름 | _선택 사항?_ | 태그 값 |
|---|---|---|
|App-Name|False|`"License-Assertion"`|
|Original|False|TXID of original upload|
|License|False|TXID of updated `License Definition`|

### 라이선스 또는 법적 도구 정의

라이선스 또는 관련 법적 도구(예: [CC0](https://wiki.creativecommons.org/wiki/CC0))를 정의하려면 다음 태그들과 함께 트랜잭션을 제출하십시오:

| 태그 이름    | _선택 사항?_ | 태그 값                                                                 |
| ------------ | ------------ | ----------------------------------------------------------------------- |
| App-Name     | True         | `"License-Definition"`                                                  |
| Logo         | True         | TXID of `License Logo`                                                  |
| Short-Name   | True         | A short (max 64 characters) 'ticker' with which to refer to the license |
| Content-Type | True         | The MIME type of the contained file, describing the license             |

트랜잭션의 **본문**은 정의되는 라이선스의 법적 텍스트를 포함해야 합니다.

### 라이선스 로고

라이선스 로고 트랜잭션은 다음과 같이 정의되어야 합니다:

| 태그 이름    | _선택 사항?_ | 태그 값                             |
| ------------ | ------------ | ----------------------------------- |
| Content-Type | False        | The MIME type of the contained file |

트랜잭션의 **본문**은 라이선스 로고의 시각적 표현을 포함해야 합니다.
