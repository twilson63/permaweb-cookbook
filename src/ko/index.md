# Permaweb로 개발하기

Permaweb 요리책은 Permaweb에서 애플리케이션을 구축하는 데 필요한 핵심 개념과 참고자료를 제공하는 개발자 자료입니다. 각 개념과 참고자료는 Permaweb 개발 생태계의 특정 측면에 초점을 맞추며 추가적인 세부사항과 사용 예시를 제공합니다.

## 개발자

과거가 블록체인에 영원히 새겨지고 미래에 무한한 가능성이 열리는 Arweave 개발 커뮤니티에 오신 것을 환영합니다. 함께 탈중앙화 웹을 구축해 봅시다!

[자세히 보기](getting-started/quick-starts/hw-cli.html)

## 기여

이 요리책은 새로운 Permaweb 개발자가 쉽게 기여할 수 있도록 설계되었습니다. 무엇을 해야 할지 모를 때라도, 요리책에 기여하는 것은 배우는 좋은 방법입니다!

모든 오픈 이슈는 여기에서 확인하세요. 기여 가이드는 여기에서 확인하세요. 요리책에 개념, 가이드 또는 참고자료가 누락되어 있다고 생각되면 이슈를 추가해 주세요.

[자세히 보기](getting-started/contributing.md)

## 요리책 읽는 방법

Permaweb 요리책은 서로 다른 목표를 가진 여러 섹션으로 나뉩니다.

| 섹션     | 설명                                            |
| -------- | ----------------------------------------------- |
| 기초     | 개발에 알아두면 좋은 Permaweb의 빌딩 블록       |
| 가이드   | 개발을 위한 다양한 도구에 대한 짧은 분량의 안내 |
| 참고자료 | 자주 필요한 코드 스니펫에 대한 참고             |
