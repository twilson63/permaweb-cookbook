# 去中心化計算

協定本身並未直接內建任何執行計算的機制——然而，這並不表示開發者無法建立並儲存在 Arweave 上的智慧合約。

[AO](https://ao.arweave.net) 是一個建立在 Arweave 之上的去中心化運算網路。透過使用 Arweave 作為儲存智慧合約狀態與使用者互動的狀態層，可以建構複雜的去中心化應用程式。

說明 AO 的運作原理與如何使用它來開發應用程式並非本手冊的範圍，但若想進一步學習，仍有大量資源可供入門。
