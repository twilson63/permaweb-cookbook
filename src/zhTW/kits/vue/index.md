# Vue 入門套件

Vue.js 是一個漸進式的 JavaScript 框架，用於構建使用者介面。與其他框架不同，它在執行時將模板編譯為 JavaScript，導致較小的檔案大小並提升效能。Vue 非常適合構建高效能且可擴展的單頁應用程式，因此在前端開發者中相當受歡迎。

Vue 入門套件 指南：

**注意：** - 由於 `npm init vue@latest` 已經使用 Vite，我們沒有為 Vue 包含 Vite 指南。

- [Create Vue App](./create-vue.md) - 使用 Create Vue 高效地建立基於 Vue.js、使用 TypeScript 和 Vite 的現代 permaweb 應用程式

::: info Permaweb 應用程式限制

- 100% 前端應用程式（不含伺服器端後端）
- 應用程式是從子路徑提供（https://[gateway]/[TX_ID]）
  :::
