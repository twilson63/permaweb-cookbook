# Vue 起始套件

Vue.js 是一個漸進式的 JavaScript 框架，可用於構建使用者介面。與其他框架不同，它會在執行時將模板編譯為 JavaScript，從而產生更小的檔案體積並提升效能。Vue 非常適合構建高效能且具可擴展性的單頁應用程式 (SPA)，因此成為前端開發者的熱門選擇。
