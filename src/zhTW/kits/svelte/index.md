# Svelte 入門套件

Svelte 是一個會編譯成 JavaScript 捆綁檔的框架，在編譯過程中會把框架本身從應用程式的發佈檔中移除。這使得其體積比其他框架小得多。Svelte 是用於 Permaweb 應用程式的理想框架。Permaweb 應用程式建立於單頁應用程式（Single Page Application）的原則，但存在於 Arweave 網路上，並由 Permaweb 閘道分發。

Svelte 入門套件 指南：

- [最小範例](./minimal.md) - 建立 Svelte Permaweb 應用程式所需的最少內容
- [Vite](./vite.md) - Svelte、TypeScript 與 Vite

::: info Permaweb 應用程式 限制

- 100% 前端應用程式（無伺服器端後端）
- 應用程式從子路徑提供服務 (https://[gateway]/[TX_ID])
  :::
