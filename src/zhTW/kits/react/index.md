# React 入門套件

React 是一個用於建立使用者介面的熱門函式庫。除了像 create-react-app 等其他流行工具之外，React 專案可以編譯成一個 bundle（打包檔）。該 bundle 可以作為交易上傳到 Permaweb，在那裡它會作為單頁應用程式 (SPA) 提供。

React 入門套件指南：

- [Vite](./turbo.md) - React + Vite，使用 `permaweb-deploy` 發佈

::: info Permaweb 應用限制

- 100% 前端應用程式（不包含伺服器端後端）
- 應用程式會從子路徑提供（https://[gateway]/[TX_ID]）
  :::
