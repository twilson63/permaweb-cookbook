# 開始使用

歡迎來到 Permaweb 食譜！本書包含概念、指南與程式片段，幫助你開始在 Arweave 上構建去中心化應用程式。

## 什麼是 Permaweb？

Permaweb 是一個全球性的社群擁有網路，任何人都可以貢獻內容或獲得報酬來維護它。它建構於 Arweave 之上，Arweave 是一個去中心化的儲存網路，可以永久存放資料。

在 Arweave 上建立與部署應用程式類似於建立傳統應用程式——主要差別在於上傳到 Arweave 的應用程式是不可變的、去中心化的，且無法被單獨移除。這些特性可能使在 Arweave 上開發充滿挑戰，但也很有價值。

本食譜旨在讓學習與開發流程更簡單。
