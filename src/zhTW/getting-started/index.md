# 入門指南

歡迎使用 Permaweb 教程手冊！本手冊包含概念、指南與程式碼片段，協助您開始在 Arweave 上構建去中心化應用程式。

## 什麼是 Permaweb？

Permaweb（永久網）是一個全球性的、由社群擁有的網路，任何人都可以為其做出貢獻或領取報酬以協助維護。它構建於 Arweave 之上，Arweave 是一個去中心化的儲存網路，用以永久儲存資料。

在 Arweave 上建立與部署應用程式的流程與建立傳統應用程式類似——主要差別在於，上傳到 Arweave 的應用程式具有不可變、更去中心化，且無法被單獨移除的特性。這些特性可能使在 Arweave 上開發具挑戰性，但也很有價值。

本手冊旨在讓學習與開發流程更簡單。
