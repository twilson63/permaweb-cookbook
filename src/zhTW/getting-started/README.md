# 在 Permaweb 上烹飪

Permaweb 食譜是提供給開發者的資源，說明在 Arweave 上構建應用程式的基本概念與參考資料。每個概念與參考都會聚焦於 Permaweb 開發生態系的特定面向，同時提供額外的細節與使用範例。

若你特別有興趣建立智慧合約，可以參考 [AO Cookbook](https://cookbook_ao.arweave.net)，該資源涵蓋建置在 AO 之上的去中心化運算平台。

## 開發者

歡迎來到 Arweave 的開發社群，這裡的過去永遠被區塊鏈記錄，而未來充滿無限可能。讓我們一起建立去中心化的網路！

[閱讀更多](welcome.md)

## 貢獻

本食譜的設計讓新的 Permaweb 開發者容易參與貢獻。即便你不會做某件事，為食譜貢獻也是學習的好方法！

在此查看所有開放議題 [here](https://github.com/twilson63/permaweb-cookbook/issues)。貢獻指南在 [here](https://github.com/twilson63/permaweb-cookbook/blob/main/CONTRIBUTING.md)。如果你發現食譜缺少某個概念、指南或參考，請提出議題。

[閱讀更多](contributing.md)

## 如何閱讀本食譜

Permaweb 食譜分成不同章節，每個章節針對不同目標。

| Section       | Description                                        |
| ------------- | -------------------------------------------------- |
| Core Concepts | Permaweb 的基本構件，對開發很有幫助                |
| Guides        | 小巧的指南，介紹不同的開發工具                     |
| References    | 常用程式片段的參考                                 |
| Starter Kits  | 前端框架啟動範本，讓你能迅速開始在 Permaweb 上開發 |
