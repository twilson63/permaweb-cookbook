# [ANS-105: ライセンスタグ](https://github.com/ArweaveTeam/arweave-standards/blob/master/ans/ANS-105.md)

著者: Sam Williams <sam@arweave.org>, Abhav Kedia <abhav@arweave.org>

## 概要

本標準は、permaweb 上の任意のトランザクションに対して、その公開ライセンスをタグ付けするための仕組みを定めます。本標準はシンプルさと使いやすさに重きを置き、permaweb エコシステム全体での幅広い採用を促進することを目的とします。

## 動機

Arweave エコシステムの目標は、すべての有益な知識と歴史の恒久的な集合的コモンズを作り、いつでも誰でも利用できるようにすることです。オリジナルのウェブと同様に、開放的な性質によりこのエコシステムはデータの再利用やコピーを促進します。従来のウェブでは、これによりライセンス情報が付与されていないまま広く共有される情報の大きなコーパスが生まれてきました。

Arweave トランザクションおよびデータエントリが提供するタグ付けシステムを簡単に利用することで、この状況の再発を避け、任意のデータに対してそのライセンスを伝達することが一般的な慣行となるウェブを構築できます。Arweave タグが関連データから分離できない性質であるため、準拠するすべてのトランザクションのライセンス情報は _アトミックに_ 利用者へ伝達されます — データへのリンクを伝達することは可能でも、そのライセンス情報を同時に送らないことは不可能です。Arweave のタグ付けシステムの性質により、これはユーザー体験に対して（ブラウザでデータがレンダリングされる場合など）強制的または目立った影響を与えることなく実現されます。

## 仕様

ライセンスタグのフォーマットは、Arweave ネットワーク上の任意のトランザクションに付与される単一の追加タグと、ライセンスタイプを定義するための 2 つの追加トランザクションフォーマットで構成されます。

### トランザクションデータへのライセンスの割り当て

トランザクションのデータを特定のトランザクションフォーマットで公開するには、次のタグを追加してください:

| タグ名  | オプション? | タグ値                                         |
| ------- | ----------: | ---------------------------------------------- |
| License |        必須 | `License Definition` の TXID                   |
| Title   |        任意 | 作品のタイトル                                 |
| Creator |        任意 | 作品の作成者の名前／識別子                     |
| Source  |        任意 | 素材を取得したソースへのリンク（該当する場合） |

### ライセンスの主張

既にアップロード済みのトランザクションに対して特定のライセンスを主張するには、次のタグを付与してください:
| タグ名 | オプション? | タグ値 |
|---|---:|---|
|App-Name|必須|`"License-Assertion"`|
|Original|必須|オリジナルアップロードの TXID|
|License|必須|更新された `License Definition` の TXID|

### ライセンスまたは法的ツールの定義

ライセンスや関連する法的ツール（例えば [CC0](https://wiki.creativecommons.org/wiki/CC0)）を定義するには、次のタグを付与したトランザクションを送信してください:

| タグ名       | オプション? | タグ値                                                     |
| ------------ | ----------: | ---------------------------------------------------------- |
| App-Name     |        任意 | `"License-Definition"`                                     |
| Logo         |        任意 | `License Logo` の TXID                                     |
| Short-Name   |        任意 | ライセンスを参照するための短い（最大 64 文字）'ティッカー' |
| Content-Type |        任意 | 含まれるファイルの MIME タイプ                             |

トランザクションの**本文**は、定義されるライセンスの法的文言を含む必要があります。

### ライセンスロゴ

ライセンスロゴのトランザクションは次のように定義してください:

| タグ名       | オプション? | タグ値                         |
| ------------ | ----------: | ------------------------------ |
| Content-Type |        必須 | 含まれるファイルの MIME タイプ |

トランザクションの**本文**は、ライセンスのロゴの視覚的表現を含む必要があります。
