# React スターターキット

React はユーザーインターフェース構築に使われる人気のあるライブラリです。create-react-app のような他の一般的なツールと併せて、React プロジェクトはバンドルにコンパイルできます。このバンドルは Permaweb にトランザクションとしてアップロードでき、そこでシングルページアプリケーション（SPA）として提供されます。

React スターターキットガイド:

- [Vite](./turbo.md) - React + Vite、`permaweb-deploy`で公開

::: info Permaweb アプリケーションの制約

- 100% フロントエンドアプリケーション（サーバーサイドのバックエンドは不可）
- アプリケーションはサブパス（https://[gateway]/[TX_ID]）から配信されます
  :::
